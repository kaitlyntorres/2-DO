Help Page

Welcome to the Help Page! This page provides guidance on using the features of our task managing application.

1. Add a New Task

You can add new tasks to your list. Here's how to do it:

  1. Fill out the task details in the form provided:
    1. Title: The title of your task to complete.
    2. Description: Give more context to your task by adding details and notes related to it.
    3. Date and Time: This is the date and time that your task needs to be comeplete by.
    4. Tag: Tags help you categorize your tasks to an overarching responsibility that you have.
    5. Priority: This is how important you task is that you need to complete. It can have a "Low", "Medium", and "High" priority.
  2. Select the "Priority" level from the dropdown list (Low, Medium, High).
  3. Click the "Add Task" button to create a new task with the specified details.

2. Editing a Task

You can edit tasks that you have added to your list. Here's how to do it:

  1. Hit the "Edit Task" button.
  2. Edit the details of the task that you would like to change.
  3. Click the "Update Task" button to finalize your changes.

3. Search Bar

The search bar allows you to filter your tasks based on specific criteria. Here's how to use it:

  1. Select a search category using the dropdown list (e.g., Title, Description, Tag, Priority).
  2. Enter your search query in the input field and press Enter.
  3. The application will filter the tasks based on your search query.

4. Sorting Tasks

You can sort your tasks by clicking on the table headers. Here's how sorting works:

  1. Click on any table header (e.g., ID, Date and Time, Title) to sort tasks in ascending order.
  2. Click on the same header again to sort tasks in descending order (Work in Progress).

5. Marking Tasks as Complete/Incomplete

The status of your tasks is displayed using checkboxes. Here's how it works:

  1. A checked checkbox indicates that a task is complete (True).
  2. An unchecked checkbox indicates that a task is incomplete (False).
  3. Click on a checkbox to toggle the status (complete or incomplete).

6. Help Button

The "Help" button on the top-right allows you to access this help page for guidance on using the application.
